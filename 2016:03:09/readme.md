## Dithering

因為早期圖像資料格式的緣故，無法儲存圖片中的全部色彩，因此在有限的儲存空間中，必須犧牲掉出現頻率較少的色彩。由於能夠顯示的色彩較少，直接影響到了肉眼在觀看圖片的感受，dithering 便是為了減緩其中的不自然性所衍生的應用技術。
